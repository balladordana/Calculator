# Калькулятор на Java

## Обзор
Этот проект реализует простой калькулятор с базовыми арифметическими операциями:
- Сложение
- Вычитание
- Умножение
- Деление

В проекте также используются юнит-тесты с JUnit для проверки корректности работы программы.

## Требования
- Java 8 или выше
